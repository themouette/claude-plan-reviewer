## v0.5.0 Requirements (All Complete)

### Heartbeat

- [x] **HB-01**: Server exposes `GET /api/ping` returning 200 OK — Validated Phase 12, 2026-05-07
- [x] **HB-02**: Frontend polls `/api/ping` every 5s, requiring 3 consecutive failures before declaring server offline — Validated Phase 13 (`useHeartbeat` + reducer), 2026-05-07
- [x] **HB-03**: Polling pauses when the browser tab is hidden and resumes on visibility — Validated Phase 13 (visibility-aware useEffect), 2026-05-07
- [x] **HB-04**: Each ping request uses `AbortSignal.timeout(3000)` to prevent hanging fetches — Validated Phase 13 (`AbortSignal.timeout(3000)` + AbortController composition), 2026-05-07

### Offline UX

- [x] **OFX-01**: When offline is detected, a persistent non-dismissable banner appears — Validated Phase 14 (`OfflineBanner` component with amber `--color-banner-bg`), 2026-05-07
- [x] **OFX-02**: When offline, submit buttons are replaced with a single "Copy to clipboard" button — Validated Phase 14 (three label ternaries in App.tsx using `offlineLabels.ts`), 2026-05-07

### Clipboard Export

- [x] **CLB-01**: Clipboard export serializes annotation state as `{"behavior":"allow"}` or `{"behavior":"deny","message":"..."}` — same format the server returns — Validated Phase 15 (`buildClipboardPayload` TDD pure function), 2026-05-07
- [x] **CLB-02**: After clipboard copy, a distinct confirmation screen says "Copied to clipboard — paste into Claude" — Validated Phase 15 (`ClipboardConfirmationView` sub-component, no auto-close), 2026-05-07

### Slash Command

- [x] **SLC-01**: `annotate.md` Step 4 updated — if no stdout result received, Claude asks user to paste the clipboard JSON into the conversation — Validated Phase 16 (annotate_content Step 4 updated in `src/integrations/claude.rs`), 2026-05-07
