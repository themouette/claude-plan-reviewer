**Cargo.toml changes:**

1. Add two new dependencies to `[dependencies]`:

```toml
self_update = { version = "0.44", default-features = false, features = ["archive-tar", "compression-flate2", "rustls"] }
dialoguer = "0.12"
```

Rationale for self_update features: `default-features = false` avoids native-tls (OpenSSL dependency that breaks musl builds). `rustls` is a pure-Rust TLS implementation. `archive-tar` + `compression-flate2` supports `.tar.gz`. Per D-09 and RESEARCH.md.

2. Add `unix-archive = ".tar.gz"` to the existing `[workspace.metadata.dist]` section. Place it after `install-path`. This changes cargo-dist's default from `.tar.xz` to `.tar.gz` â€” REQUIRED because self_update 0.44 does not support xz decompression. Per RESEARCH.md Pitfall 1.

**install.sh changes:**

3. Line 52: Change `ARCHIVE_NAME="${BINARY}-${LATEST_TAG}-${TARGET}.tar.xz"` to `ARCHIVE_NAME="${BINARY}-${LATEST_TAG}-${TARGET}.tar.gz"`

4. Line 62: Change `tar -xJf` to `tar -xzf` (J = xz, z = gzip)

5. Line 90: Change `"${INSTALL_DIR}/${BINARY}" install` to `"${INSTALL_DIR}/${BINARY}" install claude` â€” per D-07/D-08, the install.sh runs in a piped (non-TTY) context, so it MUST pass an explicit integration name. Without this change, the refactored install command would error with "No integrations specified" per D-08.

```rust
use std::path::PathBuf;
use std::fmt;

#[derive(Debug, Clone, PartialEq)]
pub enum IntegrationSlug {
    Claude,
    Opencode,
    Codestral,
}

impl IntegrationSlug {
    pub fn as_str(&self) -> &'static str {
        match self {
            Self::Claude => "claude",
            Self::Opencode => "opencode",
            Self::Codestral => "codestral",
        }
    }

    pub fn from_str(s: &str) -> Option<Self> {
        match s.to_lowercase().as_str() {
            "claude" => Some(Self::Claude),
            "opencode" => Some(Self::Opencode),
            "codestral" => Some(Self::Codestral),
            _ => None,
        }
    }

    pub fn all() -> &'static [IntegrationSlug] {
        &[Self::Claude, Self::Opencode, Self::Codestral]
    }
}

impl fmt::Display for IntegrationSlug {
    fn fmt(&self, f: &mut fmt::Formatter<'_>) -> fmt::Result {
        write!(f, "{}", self.as_str())
    }
}
```
