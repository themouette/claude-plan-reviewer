## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| ui/dist/ assets -> binary | Embedded at compile time; trusted build artifact |
| browser -> SPA | All requests are localhost; same-origin policy applies |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-01 | Tampering | Embedded assets | accept | Assets are embedded at compile time from ui/dist/; the build pipeline is under developer control. Supply chain risk is standard npm/crate ecosystem risk. |
| T-04-02 | Information Disclosure | SPA fallback | accept | FallbackBehavior::Ok returns index.html for unknown paths. This is the intended SPA behavior. No sensitive server paths are exposed -- only /api/plan and /api/decide exist. |
| T-04-03 | Spoofing | settings.json matcher | mitigate | Documentation explicitly warns that matcher MUST be "ExitPlanMode" exactly. An empty matcher fires on all permission requests, which would break normal Claude Code workflow. |
</threat_model>
