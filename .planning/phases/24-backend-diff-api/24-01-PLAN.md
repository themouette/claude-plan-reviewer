Get filename from `delta.new_file().path().map(|p| p.to_string_lossy().into_owned()).unwrap_or_default()`. Compute `previous_filename`: `Some(delta.old_file().path()...)` only when `delta.status() == git2::Delta::Renamed`, otherwise `None`.
