| T-21-03 | Information Disclosure | getElementCharOffset target not in container | mitigate | Function explicitly handles the not-contained case by returning the total container length (documented fallback). Tests cover this path so a future refactor cannot silently regress to throwing or returning NaN. |
| T-21-04 | Tampering | document mousedown click-outside listener | mitigate | AnnotationForm scopes its document listener to its component lifecycle (`useEffect` with cleanup). No global state mutation outside `onCancel`. |
