<objective>
Remove 3 stale scroll-listener tests from `ui/src/reviewer-v2/CommentPane.test.ts` that assert behaviors never implemented.

Purpose: CommentPane was built using ResizeObserver (not a scroll listener) because `rangeFromOffsets` computes scroll-invariant anchor Y coordinates — both `rangeRect.top` and `containerRect.top` shift together when the main container scrolls, so scroll events are unnecessary. The 3 failing tests were written for an earlier scroll-listener design that was superseded before implementation. They test for `addEventListener('scroll'`, `passive: true`, and `removeEventListener` — none of which exist in CommentPane.tsx. These tests are factually wrong for the current implementation and must be deleted.

- `cd ui && npm test -- --run` exits 0 with no failed tests.
