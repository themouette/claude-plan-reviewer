### Auto-fixed Issues

**1. [Rule 1 - Clippy] Collapsed nested `if` guards in commit_to_dto**
- **Found during:** Task 1 — cargo clippy -- -D warnings reported `collapsible_if`
- **Issue:** Four nested `if` statements for `peel_to_commit() + SHA match + shorthand() + is_branch()` triggered clippy's `-D warnings` gate
- **Fix:** Rewrote as a `let`-chain: `if let Ok(peeled) = r.peel_to_commit() && peeled.id() == commit_sha && let Some(name) = r.shorthand()` with a plain `if r.is_branch() / else if r.is_tag()` inner block
- **Files modified:** src/diff_api.rs
- **Commit:** de07ea4 (included in the task commit after the fix)
