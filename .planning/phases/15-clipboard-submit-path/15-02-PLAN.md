## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Annotation state → clipboard → Claude conversation | User-authored text flows from React state to system clipboard; no external input enters the app in this direction |
| navigator.clipboard API | Browser permission-gated; calling without user gesture voids transient activation in some browsers — handled by synchronous call before any await |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-15-03 | Denial of Service | navigator.clipboard.writeText | accept | Failure is caught and silenced; appState still transitions to clipboard_confirmed so the user sees the confirmation screen regardless; they can copy manually |
| T-15-04 | Information Disclosure | clipboard content | accept | User wrote the annotation text; clipboard is local-only; no network transmission |
