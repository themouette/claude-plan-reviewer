# Phase 07.2 Plan 03: Version Tracking and Post-Update Integration Refresh Summary

**One-liner:** OpenCode plugin gains version comment; update.rs detects stale Claude/Gemini/OpenCode integrations by manifest/comment and rewrites files after binary replacement.

## What Was Built

### Task 1: Version comment in OpenCode plugin (commit 20e0eb2)

- Added `// plan-reviewer-version: __PLAN_REVIEWER_VERSION__` as line 2 of `opencode_plugin.mjs`
- Modified `opencode.rs` install() to chain `.replace("__PLAN_REVIEWER_VERSION__", env!("CARGO_PKG_VERSION"))` after the existing `__PLAN_REVIEWER_BIN__` replacement
- Added `pub(crate) fn read_mjs_version(plugin_path: &Path) -> Option<String>` that parses the version comment line
- Made `opencode_plugin_path` `pub(crate)` for use by update.rs
- Made `claude_plugin_dir` `pub(crate)` for use by update.rs (noted in plan's important_note)
- Added 4 new tests: `install_writes_version_comment`, `read_mjs_version_extracts_version`, `read_mjs_version_returns_none_without_comment`, `read_mjs_version_returns_none_for_missing_file`

### Task 2: Post-update integration refresh in update.rs (commit b9ff86c)

- Added `pub fn refresh_integrations()` — reads HOME env var, calls testable core
- Added `fn refresh_integrations_with_home(home, current_version)` — detects installed integrations and rewrites stale ones
- Added `fn read_manifest_version(manifest_path)` — extracts `"version"` from JSON manifest files
- Added `fn write_claude_plugin_files(home, version)` — rewrites plugin.json and hooks.json
- Added `fn write_gemini_extension_files(home, version)` — rewrites gemini-extension.json and hooks.json
- Added `fn write_opencode_plugin_file(home, version)` — rewrites .mjs with both placeholders replaced
- Called `refresh_integrations()` inside `perform_update()` after "Successfully updated" println, before `clear_update_cache()`
- Added 10 tests covering: manifest version parse (valid/missing/invalid), Claude stale/current, Gemini stale/current, OpenCode stale/current, absent integrations no-op
