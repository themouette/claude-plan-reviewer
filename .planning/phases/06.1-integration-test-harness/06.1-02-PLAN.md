3. Create `tests/integration/install_uninstall.rs` with the following tests. Use `assert_cmd::prelude::*` and `predicates::prelude::*`. Every test creates its own `tempfile::TempDir` for HOME isolation.

   **Test functions to implement:**

   a. `help_includes_port_and_no_browser_flags` — Run `plan-reviewer --help` and assert stdout contains both `--no-browser` and `--port`. This validates success criteria #4 from the phase (both flags visible in help). No HOME isolation needed.

   b. `install_claude_creates_settings_in_isolated_home` — Run `plan-reviewer install claude` with `.env("HOME", home.path())`. Assert exit 0. Assert stdout contains "ExitPlanMode hook installed". Read `{home}/.claude/settings.json`, parse as JSON, assert `hooks.PermissionRequest[0].matcher == "ExitPlanMode"`.

   c. `install_gemini_creates_settings_in_isolated_home` — Run `plan-reviewer install gemini` with `.env("HOME", home.path())`. Assert exit 0. Assert stdout contains "BeforeTool hook installed". Read `{home}/.gemini/settings.json`, parse as JSON, assert the BeforeTool array has an entry whose `hooks[]` array contains `name: "plan-reviewer"`.

   d. `install_claude_is_idempotent` — Run `plan-reviewer install claude` twice with same HOME. Assert both exit 0. Read settings.json, parse JSON, count entries where `matcher == "ExitPlanMode"`, assert count == 1.

   e. `install_gemini_is_idempotent` — Same pattern as (d) but for gemini. Count entries in BeforeTool where hooks[].name == "plan-reviewer", assert count == 1.

   f. `uninstall_claude_on_clean_system_exits_zero` — Run `plan-reviewer uninstall claude` with fresh tmpdir HOME (no settings file). Assert exit 0. Assert stdout contains "nothing to uninstall" or "no settings file".

   g. `uninstall_gemini_on_clean_system_exits_zero` — Same pattern for gemini.

   h. `uninstall_claude_after_install_removes_hook` — Install claude, then uninstall claude, same HOME. Read settings.json, parse JSON, assert `hooks.PermissionRequest` array has no entry with `matcher == "ExitPlanMode"`.

   i. `uninstall_gemini_after_install_removes_hook` — Install gemini, then uninstall gemini, same HOME. Read settings.json, assert BeforeTool array has no entry with plan-reviewer hooks.

   **Implementation notes:**
   - Use `Command::cargo_bin("plan-reviewer").unwrap()` from `assert_cmd::prelude::*` (this re-exports `assert_cmd::Command` functionality via the `CommandCargoExt` trait on `std::process::Command`).
   - All assertions use `.assert().success()` and `predicates::str::contains(...)` for stdout checks.
   - For JSON validation after install, use `std::fs::read_to_string` + `serde_json::from_str::<serde_json::Value>`.
   - Ensure the `home` TempDir variable lives until after all assertions (Rust RAII drop — do NOT call `drop(home)` early).
