| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-18-04 | Spoofing | onAction callback target | accept | The callback is supplied by `ContentPane` which is itself inside the trust boundary. No cross-origin risk. |

| T-18-SC | Tampering | npm package installs | accept | No new packages installed. |
